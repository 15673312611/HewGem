# 项目名称
ai数字人商用系统

## ✨ 功能特性

- 口播数字人合成 文本合成和音频合成
- 形象市场 用户自由挑选
- 声音克隆功能
- 文案生成功能 支持添加参考文案库
- 邀请功能
- 分站功能
- ...

## 🚀 演示站点

体验完整功能请访问演示站: [https://ai.cutb.cn](https://ai.cutb.cn)
